  → Verificar Python: python --version
  → Reinstalar: pip install -r requirements.txt

  → Verificar dependencias: pip list
  → Leer README.md sección Troubleshooting

  → Windows: Reinstalar Python con tcl/tk
  → Linux: sudo apt-get install python3-tk
  → Mac: brew install python-tk

Error con datos:
  → Verificar permisos de carpeta data_storage/
  → Hacer backup: copy data_storage/ data_storage.bak
