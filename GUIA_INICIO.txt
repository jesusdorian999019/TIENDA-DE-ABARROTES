GUIA DE INICIO RAPIDO - FERRETERIA POS
=====================================

OPCION 1: LA MAS FACIL (RECOMENDADA)
===================================

1. En la carpeta raiz, haz DOBLE CLIC en:
   Ejecutar.bat

2. Se abrira la aplicacion automaticamente
3. Ingresa las credenciales:
   Usuario: admin
   Contraseña: admin123

LISTO!


OPCION 2: SOLO EL EJECUTABLE
============================

1. Ve a la carpeta: ferreteria_app/dist/
2. Haz DOBLE CLIC en: FerreteriaPOS.exe
3. Si pide acceso, haz clic en "Ejecutar"
4. Ingresa las credenciales (ver arriba)

LISTO!


OPCION 3: DESDE LA TERMINAL (CMD o PowerShell)
==============================================

1. Abre CMD o PowerShell en la carpeta del proyecto
2. Ejecuta:
   ferreteria_app\dist\FerreteriaPOS.exe

3. Ingresa las credenciales (ver arriba)

LISTO!


REQUISITOS DEL SISTEMA
======================

- Windows 10 o 11
- 512 MB de RAM (minimo)
- 100 MB de espacio en disco
- NO NECESITA Python instalado
- NO NECESITA conexion a Internet


CARACTERISTICAS
===============

- Gestion de inventario
- Registro de ventas
- Reportes y analisis
- Exportacion a Excel
- Sistema de usuarios
- Almacenamiento local seguro


PROBLEMAS COMUNES
=================

Problema: No aparece la ventana
Solucion: Espera 5-10 segundos, es la primera ejecucion

Problema: Antivirus bloquea la app
Solucion: Agrega excepcion en tu antivirus

Problema: Los datos no se guardan
Solucion: Verifica que tienes permisos de escritura en la carpeta

Problema: Mensaje de seguridad de Windows
Solucion: Haz clic en "Mas informacion" y luego "Ejecutar de todas formas"


CREDENCIALES POR DEFECTO
=======================

Usuario: admin
Contraseña: admin123

Puedes cambiar la contraseña o crear mas usuarios desde la aplicacion.
