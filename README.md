# Sistema de Gestión para Ferretería

Una aplicación de escritorio completa para administrar tu ferretería de manera fácil y eficiente.

## Características

- **Gestión de Inventario**: Agrega, edita y elimina productos
- **Registro de Ventas**: Registra cada venta de forma rápida
- **Reportes y Estadísticas**: Ve tus ventas diarias, semanales y totales
- **Exportar a Excel**: Exporta tus datos para imprimirlos o compartirlos
- **Alertas de Stock**: Te avisa cuando un producto está por terminarse

## Requisitos

- Windows 10 o superior
- No necesita instalación de bases de datos
- No necesita conexión a internet

## Cómo Usar

### 1. Ejecutar el Programa
Simplemente haz doble clic en **FerreteriaPOS.exe** y espera a que abra.

### 2. Iniciar Sesión
La primera vez usa:
- **Usuario:** admin
- **Contraseña:** admin123

### 3. Agregar Productos
1. Ve a la pestaña "Inventario"
2. Oprime "Agregar Producto"
3. Llena los datos:
   - Nombre del producto
   - Categoría (ej: Herramientas, Pinturas, Fijaciones)
   - Código (opcional)
   - Proveedor (opcional)
   - Precio de compra
   - Precio de venta
   - Cantidad en stock
   - Stock mínimo (para alertas)
4. Oprime "Guardar"

### 4. Registrar una Venta
1. Ve a la pestaña "Ventas"
2. Selecciona el producto de la lista
3. Ingresa la cantidad
4. Oprime "Registrar Venta"
5. ¡Listo! El stock se descuenta solo

### 5. Ver Reportes
1. Ve a la pestaña "Reportes"
2. Elige ver:
   - **Dashboard**: Resumen general con ventas de hoy
   - **Inventario**: Tus productos y valores
   - **Ventas**: Gráfico de ventas por días
   - **Productos**: Los productos más vendidos

### 6. Exportar Datos a Excel
1. En la pantalla principal, oprime "Exportar a Excel"
2. Elige dónde guardar el archivo
3. Se crea un archivo con tus datos

### 7. Resetear Reportes (borrar ventas)
1. Ve a "Reportes"
2. Oprime "Resetear Reportes"
3. Ingresa tu usuario y contraseña
4. Confirma

## Notas Importantes

- Los datos se guardan automáticamente
- No necesitas guardar manualmente
- Todo está en la carpeta "data_storage"

## Problemas Comunes

**El programa no abre**
- Asegúrate de tener Windows 10 o superior
- Descarga la aplicación nuevamente

**No puedo iniciar sesión**
- Usuario: admin
- Contraseña: admin123

**Los datos no aparecen**
- Oprime el botón "Actualizar"
